O **Opiny MCP Server** permite que assistentes de IA (como Claude Desktop, Cursor, etc.) conectem-se diretamente à sua conta Opiny para ler dados, analisar tendências e resumir feedbacks.

## O que é possível fazer?

Aqui estão alguns exemplos de como interagir com seus dados usando linguagem natural:

* Perguntar ao Claude: *"Qual foi o sentimento médio das respostas na última semana?"*
* Pedir: *"Liste os principais problemas reportados na pesquisa de Bugs."*
* Solicitar: *"Compare o NPS deste mês com o mês passado."*

## Pré-requisitos

Para utilizar a integração MCP, você precisará de:

* Conta no plano **PRO** da Opiny.
* Chave de API gerada no painel (**Configurações > Desenvolvedor**).
* Docker instalado (para rodar o servidor localmente) ou acesso à URL do servidor hospedado.
