## Filtros de Dados

Para uma análise precisa, você pode filtrar as informações exibidas no dashboard:

* **Seleção de Pesquisa:** Escolha visualizar os dados de todas as pesquisas da organização simultaneamente ou foque em um formulário específico.
* **Período de Data:** Defina um intervalo (Início e Fim) para auditar o desempenho em campanhas ou períodos específicos.

## Indicadores de Desempenho (KPIs)

No topo da página, apresentamos cinco métricas principais para leitura rápida:

<CardGroup cols={3}>

  </Card>
</CardGroup>

## Gráficos e Visualizações

### Conversão Diária
O gráfico de área compara diariamente o volume de **Visualizações** (linha pontilhada) com o de **Respostas** (área preenchida). Isso permite identificar picos de tráfego e flutuações na taxa de engajamento ao longo da semana.

### Média de Sentimento (IA)
Este componente exclusivo utiliza nossos modelos de linguagem para interpretar o tom das respostas abertas.

* **Score de 0 a 100:** Uma nota quantitativa da qualidade do feedback.
* **Classificação:** O sistema categoriza o sentimento geral como Crítico, Neutro, Bom ou Excelente.

## Exportação e Integração

Seus dados são portáveis e podem ser levados para outras ferramentas de BI ou CRM:

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/exportacao.png"
/>

<Steps>
  <Step title="Exportação em CSV">
    Baixe todos os dados brutos, incluindo metadados personalizados e colunas separadas para cada pergunta da sua pesquisa.
  </Step>
  <Step title="Acesso via API">
    Usuários **PRO** podem acessar a [API pública](/integrations/api) para consumir estes mesmos dados de forma programática em JSON.
  </Step>
</Steps>

<Note>
  A exportação para CSV está disponível apenas para pesquisas individuais. Ao selecionar "Todas as Pesquisas", o botão de download ficará desabilitado para garantir a integridade da estrutura das colunas.
</Note>
