## Conceitos básicos

Aqui estão os principais termos que você encontrará ao usar a plataforma:

* **Survey (Pesquisa):** O formulário configurado no painel. Possui um ID único (UUID).
* **Widget:** A interface visual que aparece para o usuário final em seu site ou app.
* **Response (Resposta):** Os dados coletados de uma submissão de usuário.
* **Sentiment Score:** Nota de 0 a 100 gerada pela nossa IA baseada no tom do texto do usuário.
