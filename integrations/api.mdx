A API Pública da Opiny permite que você acesse seus dados de pesquisas e respostas de forma programática. Com ela, você pode conectar os feedbacks dos seus clientes a ferramentas de BI, CRMs (como Salesforce ou HubSpot), ou criar painéis personalizados.

## Arquitetura REST

Nossa API foi construída seguindo os princípios REST. Ela possui URLs orientadas a recursos, aceita requisições codificadas em JSON e retorna respostas no formato JSON padrão, utilizando códigos de status HTTP convencionais.

## Base URL

Todas as requisições devem ser enviadas para o endereço base abaixo:

[https://api.opiny.com.br/v1](https://api.opiny.com.br/v1)


## Autenticação
Para garantir a segurança dos seus dados, todas as chamadas à API exigem uma API Key. Você pode gerar e gerenciar suas chaves no painel administrativo em Configurações > Desenvolvedor.

A autenticação é feita através do cabeçalho HTTP Authorization usando o esquema Bearer.

```bash
Authorization: Bearer op_live_your_api_key_here
```
